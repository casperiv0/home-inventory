# home-inventory

A home inventory app to keep track of items in your home (food items, tech, medical supplies, etc.).

## Features

- beautiful UI
- Great UX
- Dark/light theme
- Add users to other houses
- Great sort & filter options
- ~~Import/export products~~

## Status

- New version in full development.

## Stack

- TypeScript
- Sass
- Next.js
- Prisma
- Docker
- tRPC
